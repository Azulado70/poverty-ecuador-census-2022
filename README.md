\# Poverty and Living Conditions in Ecuador: A Latent Class Analysis Based on the 2022 Census



\## Description

This repository contains the full analytical pipeline for a research paper on 

multidimensional poverty and living conditions in Ecuador, based on the 

VIII Population and Housing Census 2022 (INEC).



\## Dataset

The raw data is publicly available at:

\- \*\*Zenodo DOI:\*\* https://doi.org/10.5281/zenodo.20943520

\- \*\*Source:\*\* Instituto Nacional de Estadística y Censos (INEC) Ecuador

\- \*\*Files:\*\* CPV\_Hogar\_2022, CPV\_Población\_2022, CPV\_Vivienda\_2022



\## Methodology

1\. \*\*Principal Component Analysis (PCA)\*\* — Living Conditions Index (LCI)

2\. \*\*Latent Class Analysis (LCA)\*\* — Poverty profile identification

3\. \*\*Multinomial Logistic Regression\*\* — Determinants of poverty class



\## Results

\- 99,681 records analyzed (national random sample)

\- 7 latent classes identified (probability >95% per class)

\- LCI mean: 69.32 (SD: 15.15)
